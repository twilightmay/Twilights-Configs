## CS2 — Monthly Skin Packs

Each pack is built around a specific theme — a mood, colour palette, and visual direction. Skins, stickers, and charms are all chosen to serve that theme.

Every pack shows you at a glance:
- ✅ / ❌ **Stickers** — whether the pack includes sticker placements
- ✅ / ❌ **Gun Charms** — whether gun charms are included
- ✅ / ❌ **All Weapons** — whether every weapon slot (rifles, pistols, knives, gloves) is covered

### How to use a CS2 pack

1. Get **lunaR** from [frostchanger.com](https://frostchanger.com)
2. Download the pack config file from the site

## FiveM — Cheat Configs

Three config profiles for the **Lucid** cheat, each targeting a different playstyle:

### Legit Config
- **Risk level:** Low
- **Features:** Aimbot, Humanize, ESP, Player settings
- Human-like movement and natural aim assist. Designed to look completely organic — no obvious patterns, no triggers.

### Semi Rage Config
- **Risk level:** Medium
- **Features:** Aimbot, Silent, ESP, Player settings
- The middle ground. Assisted aim with controlled aggression. Pushes limits without going overboard.

### Rage Config
- **Risk level:** High
- **Features:** Weapon, Exploit, ESP, Player settings
- Full send. Maximum aggression tuned specifically for Lucid's rage features.

### How to use a FiveM config

1. Get **Lucid** from [frostchanger.com](https://frostchanger.com)
2. Click **Download Config** on the config you want
3. Put them into this folder : C:\Users\default\AppData\Roaming\lucid\configs
